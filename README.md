# 🎵 Spotify Layout Practice

This project is a front-end clone of Spotify’s web layout, built using HTML and CSS. It focuses on recreating the visual structure and design of a modern music streaming interface for practice purposes.

## 📄 Project Overview

A static Spotify-inspired UI that includes:
- Sidebar navigation (Home, Search, Library)
- Main content area with playlists and cards
- Header section with controls
- Clean and modern dark-themed layout

## 🛠️ Built With

- HTML5  
- CSS3  
- No JavaScript used

## 🧠 Key Learnings

- Practice building complex layouts using CSS  
- Improve positioning and alignment skills  
- Work with flexbox/grid for structured UI  
- Understand modern UI design patterns  
- Recreate real-world website layouts
